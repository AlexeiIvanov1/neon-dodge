# Neon Dodge

A tiny browser game ready for GitHub Pages.

## Run locally
Open `index.html` in a browser.

## Connect to GitHub and publish
1. Create a new empty repository in your GitHub account (`AlexeiIvanov1`), for example: `neon-dodge`.
2. In this folder, run:

```bash
git add .
git commit -m "Initial game"
git remote add origin https://github.com/AlexeiIvanov1/neon-dodge.git
git push -u origin main
```

3. On GitHub: `Settings` -> `Pages` -> Source = `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`, then save.
5. Your game URL will be:

```text
https://alexeiivanov1.github.io/neon-dodge/
```

If you choose a different repo name, replace `neon-dodge` in the commands and URL.
